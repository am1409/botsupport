FAQ

Common questions

Does the bot only answer from my documents?
Yes. The AI is strictly constrained to the content you upload. It will never make up answers or go off-topic. If a question isn't covered by your docs, it politely tells the user to contact you directly.
What file types can I upload?
You can upload PDF files or paste any public URL (your FAQ page, help centre, product pages, etc.). We extract the text, chunk it, and make it searchable for the AI.
Will it work on my website platform?
Yes. The embed code is a single JavaScript tag that works on any platform — Shopify, Webflow, WordPress, Wix, custom HTML, or anything else. If your site can run JavaScript, it works.
What happens when I hit my chat limit?
The bot displays a message directing users to contact you directly. You'll receive an email warning at 80% usage so you can upgrade before hitting the limit.
Can I cancel anytime?
Absolutely. Cancel from your dashboard at any time with no cancellation fees. Your bot will remain active until the end of the billing period.
